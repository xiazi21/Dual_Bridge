# DualBridge: A Dual-Teacher Framework Bridging HGNNs to MLPs

This repository contains the official implementation of our **SDM 2026** paper:

> **DualBridge: A Dual-Teacher Framework Bridging HGNNs to MLPs**  


DualBridge is a dual-teacher knowledge-distillation framework that transfers relation-based and meta-path-based knowledge from heterogeneous graph neural networks (HGNNs) to an efficient multilayer perceptron (MLP) student.
